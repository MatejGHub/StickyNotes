Add sticky notes to any webpage using an intuitive floating bubble interface. Perfect for content review, feedback, and collaboration.

== Description ==

Sticky Comment allows logged-in users to create sticky notes on any webpage using a floating bubble interface. Simply click the bubble and select "Add New Note", then click anywhere to place your note. These notes persist across page visits and provide an excellent way for teams to collaborate, leave feedback, or mark areas of interest.

= Features =

* **Easy Creation**: Click the floating bubble and select "Add New Note" to place notes anywhere
* **Drag & Drop**: Reposition notes by dragging the handle
* **Collapsible**: Minimize notes with the +/- toggle
* **Auto-Save**: Notes save automatically as you edit
* **Admin Dashboard**: View all notes organized by post
* **Configurable**: Set maximum notes per post and user role restrictions
* **User Feedback**: Visual notifications for all actions
* **Notes Management**: View all notes in an organized list with focus and delete options
* **Toggle Visibility**: Show/hide all notes instantly
* **Responsive**: Works on all device sizes
* **Floating Interface**: Modern bubble interface that stays out of the way

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/sticky-comment/` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Sticky Comments screen to configure the plugin
4. Look for the floating bubble in the bottom-right corner of any page
5. Click the bubble and select "Add New Note" to start creating notes

== Frequently Asked Questions ==

= How do I create a sticky note? =

Click the floating bubble in the bottom-right corner, select "Add New Note", then click anywhere on the webpage to place your note. You can then edit and move it around.

= Can I limit who can create notes? =

Yes! Go to Sticky Comments in your admin menu and set the minimum user role required in the Settings tab.

= How many notes can I create? =

By default, 10 notes per post. This can be configured in the settings from 1-50 notes.

= Do notes work on all post types? =

Yes, sticky notes work on posts, pages, and custom post types.

= Are notes private? =

Notes are visible to all logged-in users who have permission to create notes.
